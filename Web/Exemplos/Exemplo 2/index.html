Introdução ao CSS

Estamos aprendendo sobre o CSS e o HTML. Em como fazer os estilos das páginas web.CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Com o CSS, você pode controlar o layout, as cores, as fontes e muito mais. Você pode aplicar estilos a elementos HTML usando seletores, propriedades e valores.
